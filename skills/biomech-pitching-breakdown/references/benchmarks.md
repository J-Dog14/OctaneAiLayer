# Benchmarks & Status Bands

Use these ranges to categorize each metric. Don't invent your own — these are calibrated against published norms (Driveline, ASMI, KMotion) AND the 8ctane cohort.

## Status terminology

| Status | Meaning | Color (hex) |
|---|---|---|
| Elite | At or above the top of elite range / cohort best | 548235 (dark green) |
| Strength / Good | Within elite range | 70AD47 (light green) |
| Watch | Within reasonable range but room to improve | ED7D31 (orange) |
| Flag / Below | Outside elite range, needs attention | C00000 (red) |
| Neutral | No clear signal | 7F7F7F (gray) |

| Range | Status |
|---|---|

| Range | Status |
|---|---|
| ≤ 1.2 Nm/mph | Healthy |
| 1.2–1.35 | Watch |

| Range | Status |
|---|---|

| < 0.5 BW | Likely sub-max capture (re-test recommended) |

## Reference ranges for non-cylinder context metrics

### Peak segment angular velocities (transverse)

| Segment | Elite | MLB range |
|---|---|---|
| Pelvis | 770–900 °/s | — |
| Thorax | 1,200–1,500 °/s | — |
| Humerus | — | 4,500–6,000 °/s |
| Hand | — | 4,500–6,000+ °/s |

### Amplification ratios (chain analysis)

| Transition | Elite | What "below" means |
|---|---|---|
| Pelvis → Thorax | 1.4–1.8× | Trunk under-amplifies — energy leak at trunk |
| Thorax → Humerus | 4–6× | Note: convention-dependent, less useful as a flag |
| Humerus → Hand | 0.9–1.0× (typical) | > 1.0× means arm is rescuing chain (compensation) |

### Pelvis decel timing (PelvisRot_Stop − FS)

| Range | Status |
|---|---|
| < 140 ms | Elite (early stop, clean transfer) |
| 140–180 ms | Typical |
| > 180 ms | Late (chain hasn't fully transferred) |

### External rotation at FS (arm timing)

| Range | Status |
|---|---|
| < 25° | Late arm timing (associated with hard throwers) |
| 25–40° | Typical |
| 40–60° | Arm-early |
| > 60° | Severely arm-early (red flag, esp. with low MER) |

### Max external rotation (MER)

| Range | Status |
|---|---|
| 170–190° | Elite |
| 190–200° | Slightly above band (verify safety) |
| < 170° | Below band (limited layback) |
