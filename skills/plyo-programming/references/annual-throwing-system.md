# 8ctane Annual Throwing System — Phase Calendar

8ctane pitching development follows a periodized yearly system. Plyo intensity and throwing workload must align with the annual development calendar — the day-level plyo cadence in `cadence-rules.md` is **always subject to the phase guardrails defined here**.

## Why this matters

Skipping or rushing through phases (especially Rebuild Capacity and Workload Ramp) increases:

- Injury risk
- Fatigue accumulation
- Inconsistent velocity gains

Athletes need to progress gradually. The phase you're in determines what plyo work is appropriate regardless of how the athlete feels day-to-day.

## Core principles

1. Throwing intensity follows annual periodization — never random.
2. Plyo phases must match throwing workload. Plyo intensity tracks throwing intensity, not the other way around.
3. Velocity development happens far from competition.
4. Recovery phases are required for long-term durability.
5. Each phase progresses gradually; deloads come often.
6. Drills supplement and prime — they never replace throwing.
7. The athlete must understand the *concept* of every drill they're doing.

## Phase Calendar Quick Reference

| Phase | Typical Timeframe | Primary Focus | Throwing Frequency | Plyo Intensity |
|---|---|---|---|---|
| Recovery | 3–5 weeks (Jul–Aug) | Arm health + tissue recovery | 2–3×/week | None / +0 only |
| Rebuild Capacity | 4–6 weeks (Aug–Sep) | Restore arm health + patterns | 3–4×/week | +0, +1 |
| Workload Ramp | 4 weeks (Sep–Oct) | Increase throwing workload | 4–5×/week | +0, +1, +2 |
| Velocity | 3–5 weeks (Nov–Dec) | Velocity + intent development | 4–5×/week + deload | +1 → +2, +3 |
| Velocity Transfer | 4 weeks (Dec–Jan) | Transfer velocity to the mound | 4–5×/week + deload | +1 → +2 |
| Preseason Build | 6–8 weeks (Jan–Feb) | Command, pitch design, bullpens | 4–5×/week + deload | +1, +2 |
| In-Season Maintenance | Season (Mar–Jul) | Maintain velocity + arm health | 3–5×/week | +0 → +1, +2 |
| Postseason Reset | 2–4 weeks (Aug–Sep) | Recovery + reassessment | 2–3×/week | +0 |

## Phase Details

### Phase 1 — Recovery

- **Duration**: 3–5 weeks
- **Purpose**: Allow arm + connective tissue to fully recover from competitive workload
- **Throwing frequency**: 2–3×/week
- **Plyo intensity**: None or +0 only
- **Hard rules**: No high-intent throwing. No pulldowns. No high-intensity plyos.

### Phase 2 — Rebuild Capacity

- **Duration**: 4–6 weeks
- **Purpose**: Re-establish throwing patterns and build arm durability
- **Throwing frequency**: 3–4×/week
- **Plyo phase**: +0, +1
- **Drill structure**: Activation → Reverse Throws → Pivot Pickoffs → P90s
- **Throwing focus**: Catch progression, arm path, movement efficiency
- **Intent level**: Low

### Phase 3 — Workload Ramp

- **Duration**: 4 weeks
- **Purpose**: Increase throwing workload, introduce moderate intent
- **Throwing frequency**: 4×/week early, 5×/week in last 2 weeks
- **Plyo phase**: +0, +1, +2
- **Drill structure**: Activation → foundational drills → pattern drills → step-behinds → light pulldowns (late phase)
- **Throwing focus**: Long toss progression, rhythm, momentum
- **Intent level**: Moderate

### Phase 4 — Velocity Phase

- **Duration**: 4–6 weeks
- **Purpose**: Develop arm speed; increase velocity ceiling
- **Throwing frequency**: 3–5×/week
- **Plyo phase**: +0, +1, +2, **+3**
- **Drill structure**: Activation → Pivot Pickoffs → Step-Behinds → Roll-Ins → possible Underload Throws
- **Throwing focus**: High intent throws, velocity tracking
- **Intent level**: High
- **Hard rules**: High intent days MUST be separated by recovery days. Monitor fatigue markers. **Record velos and video for every high-intent session.**

### Phase 5 — Velocity Transfer

- **Duration**: 4 weeks
- **Purpose**: Transfer velocity gains to mound mechanics
- **Throwing frequency**: 4–5×/week + deload
- **Plyo phase**: +0, +1, +2
- **Structure**:
  - Weeks 1–2: Throwing / Throwing / Mound / Recovery / Throwing
  - Weeks 3–4: Throwing / Mound (light) / Recovery / Throwing / Mound
- **Throwing focus**: Mound velocity, pitch shape, command

### Phase 6 — Preseason Build

- **Duration**: 6–8 weeks
- **Purpose**: Prepare for competition
- **Throwing frequency**: 4–5×/week + deload
- **Plyo phase**: +1, +2
- **Throwing focus**: Bullpens, pitch design, command development
- **Note**: High-intent throwing decreases slightly to make room for command, shape, sequencing, in-game awareness

### Phase 7 — In-Season Maintenance

- **Duration**: Full season
- **Purpose**: Maintain arm health and velocity
- **Throwing frequency**: 3–5×/week
- **Plyo phase**: +0 → +1, occasional +2
- **Typical Starter Weekly Structure**:
  - Day 0: Game
  - Day +1: Recovery
  - Day +2: Light catch
  - Day +3: Bullpen
  - Day +4: Throwing
  - Day +5: Light intent
  - Day +6: Game prep
- Plyos are used as needed to prime movement patterns (athlete's favorite drills, low volume).

### Phase 8 — Postseason Reset

- **Duration**: 2–3 weeks
- **Purpose**: Evaluate athlete progress and begin recovery
- **Plyo intensity**: +0 only
- **Athletes complete**: 8ctane Blueprint Assessment (biomechanics, force plate, pitch data, strength testing)
- **Output**: Next year's individualized development plan

## How to use this calendar in plyo programming

When deciding a plyo level for a given day:

1. Use the cadence rules (`cadence-rules.md`) to get the "candidate level" based on role + day in the week.
2. Look up the athlete's current annual phase.
3. Check whether the candidate level is permitted in that phase (table above).
4. If allowed → use it. If not → downgrade to the highest level the phase permits.

Examples:

- A starter's cadence puts them at P2 (3 days out from next start) but they're in **Rebuild Capacity** — the phase only allows +0 and +1, so the prescription is P1.
- A reliever hasn't pitched in 4 days and feels great in **Velocity Phase** — phase allows all levels, cadence supports P2, so P2 it is. P3 is permitted but should only be assigned if intent education is the genuine goal.
- It's mid-season and a starter is on Day +3 (pen day) — In-Season permits up to +2, so P2 is fine, but volume should sit at the lower end of the P2 range.

## Phase transitions

- Don't skip phases. Recovery → Rebuild → Ramp is the foundation that protects the connective tissue going into Velocity and beyond.
- Deloads come often. A deload week typically means dropping plyo intensity one level for the week (e.g., a Velocity-phase deload runs at +1 instead of +2, +3).
- When transitioning between phases, the first week of the new phase should be conservative — use the bottom end of allowed levels.
