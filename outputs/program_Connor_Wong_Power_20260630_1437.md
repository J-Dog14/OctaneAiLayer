## Plyo / Throwing — 3-day cycle · Starter · phase: Velocity Phase

> This three-day plyo cycle for Connor Wong, a college starter in the Velocity Phase, directly addresses his primary deficits in trunk lateral flexion, pelvis obliquity, humerus angular velocity, and shotput acceleration. The Velocity Phase permits high-intensity work, so no plyo level downgrades were necessary due to phase. The P0 day prioritizes tissue restoration and fundamental movement, while P1 introduces balanced rotational work and light intent. The P2 session aggressively targets core rotational power and arm speed, utilizing heavier weights and higher-intensity drills to improve hip-shoulder separation and full-body intent, which are critical for closing his observed biomechanical and Proteus gaps.
